| T-08-01 | Elevation of Privilege | catalog-search | mitigate | `requireRole("MEMBER")` first line of `searchCatalog` (T-01); not middleware |
| T-08-02 | Information Disclosure | my-profile | mitigate | Profile renders ONLY the session user's own record (`getSession` → own id); no `params`-driven lookup of arbitrary users (IDOR prevented) |
| T-08-03 | Tampering | search query injection | mitigate | Query passed as a Prisma `contains` bound parameter (parameterized) — no `$queryRawUnsafe`; arbitrary input cannot alter the query (T-03) |
| T-08-04 | Information Disclosure | over-fetching copy data | mitigate | `searchCatalog` returns whitelisted `BookCardData` (title/author/counts) — raw BookCopy/loan rows not leaked to members |
</threat_model>
