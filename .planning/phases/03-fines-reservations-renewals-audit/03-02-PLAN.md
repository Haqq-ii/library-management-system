## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Spoofing | waiveFine | mitigate | requireRole("LIBRARIAN") is the first call — session verified server-side via Better Auth; middleware not trusted (CVE-2025-29927) |
| T-03-02-02 | Tampering | waiveFine fineId | mitigate | Zod min(1) validates fineId is non-empty; Prisma P2025 catches non-existent fineId; actorId is session.user.id, not user-supplied |
| T-03-02-03 | Tampering | waiveFine reason | mitigate | Zod min(1) validates reason is non-empty; no HTML injection risk (stored as plain string, not rendered as HTML) |
| T-03-02-04 | Information Disclosure | /fines page | mitigate | Server Component auth guard: role !== "LIBRARIAN" redirects to /dashboard before any data fetch |
| T-03-02-05 | Tampering | AuditLog.actorId | mitigate | actorId = session.user.id from requireRole() return value — cannot be client-supplied |
| T-03-02-SC | Tampering | npm/pip/cargo installs | accept | No new package installs in this plan |
</threat_model>
